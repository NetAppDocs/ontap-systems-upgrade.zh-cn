.用于MetroCluster FC配置
对于 MetroCluster FC 配置，您必须尽快更换灾难恢复 / 故障转移站点节点。不支持 MetroCluster 中的控制器型号不匹配，因为控制器型号不匹配会使发生原因灾难恢复镜像脱机。在更换第二个站点的节点时，使用命令 `skip-metrocluster-check true` 选项绕过 MetroCluster 检查。
