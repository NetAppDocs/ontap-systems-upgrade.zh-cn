= 第 6 阶段。完成升级
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
在第 6 阶段，您确认新节点设置正确。如果其中一个新节点具有统一目标适配器，则必须还原任何端口配置，并且可能需要更改适配器的个性化设置。如果新节点已启用加密，则还必须设置存储加密。您还必须停用旧节点。

. link:resume_snapmirror_ops.html["恢复 SnapMirror 操作"]
