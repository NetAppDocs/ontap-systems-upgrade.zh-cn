= 使用板载密钥管理器管理身份验证密钥
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
您可以使用板载密钥管理器(OKM)管理身份验证密钥。如果您已设置OKM、则必须先记录密码短语和备份材料、然后再开始升级。

.步骤
. 记录集群范围的密码短语。
+
这是在使用命令行界面或REST API配置或更新OKM时输入的密码短语。

. 验证密码短语和密钥管理器操作：
+
`sSecurity key-manager 板载同步`

+
出现提示时，输入上一步中记录的群集范围的 OKM 密码短语。

. 通过查看以下命令的输出来验证 cc-mode 是否已启用：
+
`security key-manager config show`

. 备份密钥管理器信息：
+
`s安全密钥管理器板载 show-backup`



.下一步是什么？
使用板载密钥管理器管理身份验证密钥后，link:quiesce_snapmirror_relationships.html["暂停 SnapMirror 关系"]。
