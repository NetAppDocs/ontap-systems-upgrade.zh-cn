= 停用 node2
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
要淘汰 node2，请正确关闭 node2，然后将其从机架或机箱中移除。

.步骤
. 恢复操作：
+
`s系统控制器更换恢复`

+
节点会自动暂停。



.完成后
升级完成后，您可以停用 node2 。请参见 link:decommission_old_system.html["停用旧系统"]。
