= 恢复 SnapMirror 操作
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您可以恢复升级前暂停的 SnapMirror 传输，并恢复 SnapMirror 关系。升级完成后，更新将按计划进行。

.步骤
. 验证目标上的 SnapMirror 状态：
+
`snapmirror show`

. 恢复 SnapMirror 关系：
+
`snapmirror resume -destination – vserver <vserver_name>`
