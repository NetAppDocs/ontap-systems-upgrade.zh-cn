= 持久启用交换机
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
您必须持久启用网络结构中的交换机。

持久启用交换机：

* 对于 Brocade 交换机，请使用 `sswitch/CfgPersistentenable` 命令。
+
[listing]
----
FC_switch_A_1:admin> switchCfgPersistentenable
----
* 对于 Cisco 交换机，请使用 `no suspend` 命令。
+
[listing]
----
vsan [vsna #]no suspend
----
