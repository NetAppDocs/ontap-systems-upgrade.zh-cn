= 停用 node2
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
要淘汰 node2，请正确关闭 node2，然后将其从机架或机箱中移除。

.步骤
. 恢复操作：
+
`s系统控制器更换恢复`

+
节点会自动暂停。



.完成后
退出节点 2 后，确认其已关闭电源并且未连接到网络。将 node2 的旧硬件保持此状态，直到完成 node1 和 node2 的升级。然后，您可以在以下情况下停用 node2link:manage-authentication-using-kmip-servers.html["第 6 阶段。完成升级"] 。
