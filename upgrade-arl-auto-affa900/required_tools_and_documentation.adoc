= 所需的工具和文档
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
要执行升级，您必须具有接地线，并且在升级过程中需要参考其他文档。

要将AFF A800升级到AFF A90或AFF A70、请确认100GbE缆线的长度至少为1米。

请参见 link:other_references.html["参考资料"] 可访问此升级所需的参考文档和参考站点列表。
